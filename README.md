# 🎂 Madison Beer · 27th Birthday Fanpage  

A clean, elegant fan page celebrating **Madison Beer’s 27th birthday** (March 5) and the release of the **Locket** album. Built with pure HTML/CSS/JS — no frameworks, just love.

---

## ✨ Features

- **🌆 Hero carousel** – rotating full‑screen photos with dynamic glow (changes with each image).  
- **⏳ NYC countdown** – ticks down to midnight March 5 (New York time).  
- **📸 Infinite photo strip** – 27+ polaroid‑style pictures scroll endlessly, pause on hover, zoom on hover.  
- **💌 Birthday wishes** – cream‑white section with link to Google Forms (add your own).  
- **🔒 Locket flip cards** – 6 interactive cards: golden lock on front, personal message on back (flip on click).  
- **🎧 Support Madison** – big Locket cover + floating past albums + streaming links (Spotify, Apple Music, YouTube, official site).  
- **📱 Fully responsive** – mobile‑optimised hero images swap under 768px.  
- **✨ Smooth scroll animations** – titles fade in as you browse.

---

## 🛠 Built with

HTML5 · CSS3 (Flexbox, Grid, animations) · Vanilla JS · Google Fonts (Inter, Imperial Script) · Font Awesome

---

## 🚀 Quick start

1. Clone or download.  
2. Place your images in `media/` (names match those in HTML).  
3. Replace placeholder links (Google Forms, socials).  
4. Open `index.html` in a browser.

To host on GitHub Pages: push to `yourusername.github.io` and enable Pages.

## 🎨 Customise

- **Glow themes** – edit `body.theme-1` … `theme-4` in CSS.  
- **Flip card texts** – change `cardData` array in `initFlipCards()`.  
- **Countdown target** – adjust month/day in `getNextBirthdayNY()` (months are 0‑indexed).
